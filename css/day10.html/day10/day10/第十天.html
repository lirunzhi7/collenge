HTML5新增标签

标签名 描述
header 定义头部
footer 定义尾部
nav 定义导航
section 定义大区块
article 定义文章
aside 侧边栏
details 文档详情 谷歌和safari浏览器支持
summary 定义details的标题 谷歌和safari浏览器支持
mark 标记 行内元素
figure 配图说明
figcaption 配图详情
audio 音频
video 视频

audio/video常用属性

  • 行内块元素
  • src 音视频的路径
  • controls 控制条的显示隐藏
  • autoplay 自动播放
  • loop == loop="loop"循环播放

HTML5新增表单类型

  • input type
    • type="text" 文本输入框
    • type="password" 密码框
    • type="radio" 单选按钮
    • type="checkbox" 多选按钮
    • type="submit" 提交按钮
    • type="reset" 重置按钮
    • type="button" 普通按钮
  • textarea 文本域
  • select 下拉列表
  • option 列表项
  • label 可以于表单元素进行关联

新增的type值

  • type=date 年月日 日期
  • number 数字
  • tel 电话
  • range 滑动条
  • email 邮件
  • time 时间

CSS3新增属性

  • border-radius 圆角
    • 值：可以为具体数值，可以为百分比
  • box-shadow 盒子阴影
    box-shadow: x轴方向阴影 y轴方向阴影 阴影模糊度 阴影的外延值 阴影颜色 inset;
